# multiclass-image-classification
This is project implement a deep learning CNN model to predict the class from which a image belongs to. The model is train on a dataset consisting of 6,899 images from 8 distinct classes compiled from various sources. The classes include airplane, car, cat, dog, flower, fruit, motorbike and person. The dataset is available at [Kaggle](https://www.kaggle.com/prasunroy/natural-images).
